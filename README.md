# LightWidget
## 项目介绍
本项目提供一个简易Qt插件框架示例

该仓库暂时停止更新，但可在[WidgetExplorer](https://github.com/howdy213/WidgetExplorer)中获取后续进展
## 环境依赖
Qt6 (MSVC2022 64bit)
## 目录结构
    │ LICENSE                        //许可证
    │ LightWidget.pro                //项目文件
    │ README.md                      //README文件
    ├─ExamplePlugin                  //示例插件
    ├─LightWidget                    //主程序
    └─WidgetExplorerSDK              //WidgetExplorer核心库
        ├─WConfig                    //配置
        ├─WDef                       //定义
        ├─WE
        ├─WFile
        └─WPlugin                    //插件加载类
